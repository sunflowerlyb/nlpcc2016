1. Files
1) train.dat
		the training files
2) test.dat
		the test files
3) dev.dat
		the develop files
4) background.txt
		the background data, from which the training and test data are drawn. The purpose is to find the more sophisticated features by the unsupervised way.


2. Description of the Task
1) Track:
		CLOSED
		SEMI-OPEN
		OPEN



The submitted result files should be encoded in UTF-8. You need choose your track when you submit the result file.

#Track should be ''CLOSED'', ''SEMI-OPEN'' or ''OPEN'' to indicate which track the result is.

The format of the submitted files should be like the format of the train files. 
The number of lines in the results must be same with the test file.
